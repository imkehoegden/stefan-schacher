# Website für den Fotografen Stefan Schacher

Eine moderne Portfolio-Website für den Leipziger Fotografen **Stefan Schacher**. Die Seite dient dazu, seine Arbeiten stilvoll zu präsentieren und potentiellen Kund:innen einen umfassenden Eindruck seiner fotografischen Arbeit zu vermitteln.

---

## Features

- Minimalistisches & visuell fokussiertes Design
- Schnelle Ladezeiten dank Vite
- Styling mit Tailwind CSS
- Vollständig responsive (Mobile, Tablet, Desktop)
- Galerien, Slider und Lightboxes zur optimalen Präsentation der fotografischen Arbeiten
- Kontaktmöglichkeiten für Anfragen

---

## Tech Stack

- **Frontend:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS

---

## Getting Started

### Voraussetzungen

- Node.js
- npm

### Installation

```bash
# Repository klonen
git clone https://github.com/imkehoegden/website-stefan-schacher.git

# In das Projektverzeichnis wechseln
cd website-stefan-schacher

# Dependencies installieren
npm install
